# Set up React project

1. Create a new Vite project (React + TypeScript)

```bash
npm create vite@latest my-app -- --template react-ts
```

2. Add Prettier

```json
{
    "scripts": {
        "format": "prettier --write \"src/**/*.{js,jsx}\""
    },
}
```

3. Add ESLint

4. Add `.gitignore`

5. Add TanStack Router
